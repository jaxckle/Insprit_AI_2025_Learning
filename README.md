# Insprit_AI_2025_Learning

Welcome to **Insprit_AI_2025_Learning**, a personal AI/ML learning repository by Jaxon! This repo contains hands-on experiments and tutorials with machine learning and deep learning techniques, using Python and Jupyter notebooks.

## Contents
| Notebook | Description |
|----------|-------------|
| `Jaxon_LinearRegression.ipynb` | Step-by-step exploration of linear regression, including dataset preparation, model training, and evaluation. |
| `Jaxon_LogisticRegression.ipynb` | Implementation of logistic regression for classification problems with practical examples. |
| `Jaxon_NeuralNetworks.ipynb` | Introduction to building and training basic neural networks from scratch. |
| `Jaxon_Student_Convolutional_Neural_Networks.ipynb` | Applying convolutional neural networks (CNNs) for image classification tasks. |

## Technologies Used
- Python 3.x  
- Jupyter Notebook  
- NumPy, Pandas, Matplotlib  
- Scikit-learn  
- TensorFlow / Keras  

## Goals
- Document hands-on AI/ML learning and experiments.  
- Provide tutorials and examples for linear models, neural networks, and deep learning.  
- Serve as a foundation for future AI projects and applications.
